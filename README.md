# Alfox.ai Certificate Portal

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Set environment variables (recommended):
   - `ADMIN_PASSWORD` - password for admin (default: alfoxadmin)
   - `SESSION_SECRET` - session secret

3. Start:
   ```
   npm start
   ```

4. Open `http://localhost:3000/`

Admin login: visit `/admin/login` and use the admin password.

Seeded sample certificate: `ALX-2025-001` for Lucky KN.
